Rising Sun: an Interactive Java Program



A simple program that prompts the user to click repeatedly making the sun rise higher, and higher after each click. If the user has not started clicking the text instructions will be shown....

After the user begins clicking the instructions will be hidden, If the user's mouse pointer leaves the program window both the sun and text instructions will be reset.



Rising Sun uses some key Java features to achieve this behavior.


Mouse Events:

  • onMouseClick(Location point)

Once the mouse is clicked, the sun moves, and instructions are hidden.

  • onMouseExit(Location point)

Once the mouse exits the program window, return sun to initial location, and show instructions.
